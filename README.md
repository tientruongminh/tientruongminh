## About me

I build AI systems that connect research ideas to real infrastructure: data pipelines, backend services, training workflows, model deployment, and product-facing interfaces.

Current direction: **RF sensing, WiFi CSI, machine learning systems, and production-grade AI platforms**.

## Featured work

### RF-WorldPose

A production/research platform for WiFi CSI human sensing and WiFi-only skeleton/DensePose inference.

```text
ESP32-S3 CSI nodes
   -> Rust edge gateway
      -> NATS + S3 data lake
         -> Bronze/Silver/Gold ETL
            -> Helios GH200 training
               -> ONNX/Triton inference
```

Repository: [`tientruongminh/rf-worldpose`](https://github.com/tientruongminh/rf-worldpose)
